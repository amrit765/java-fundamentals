# Operators & Conditional Statements ⚙️

## 📌 Description

These concepts are critical for writing logical programs and form the base of problem-solving in Data Structures & Algorithms.

---

## 📚 Topics Covered

### 🔹 Operators
- Arithmetic Operators (+, -, *, /, %)
- Unary Operators (++, --)
  - Pre-increment / Pre-decrement
  - Post-increment / Post-decrement
- Relational Operators (==, !=, >, <, >=, <=)
- Logical Operators (&&, ||, !)
- Assignment Operators (=, +=, -=, *=, /=)

### 🔹 Conditional Statements
- if statement
- if-else statement
- else-if ladder
- Nested conditions
- Ternary operator
- switch statement
