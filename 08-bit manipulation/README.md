# Bit Manipulation ⚡
## 📌 Description

Bit manipulation enables solving problems at the binary level, offering highly optimized solutions in terms of time and space. It is widely used in competitive programming and system-level optimization.

---

## 📚 Topics Covered
### 🔹 Bitwise Operators
AND (&),
OR (|),
XOR (^),
NOT (~),
Left Shift (<<),
Right Shift (>>)
### 🔹 Bit Operations
Get i-th bit
Set i-th bit
Clear i-th bit
Update i-th bit
## 💻 What I Practiced
### 🔸 Basic Problems
- Check if a number is odd or even using bitwise AND
### 🔸 Intermediate Problems
- Clear last i-th bits
- Clear range of bits
### 🔸 Advanced Problems (High Value)
- Count set bits
- Fast exponentiation using binary approach
