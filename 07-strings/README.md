# Strings in Java 🔤
## 📌 Description

Strings are essential for handling textual data. Mastery of string manipulation is critical for solving real-world problems, especially in pattern matching, parsing, and optimization tasks.

---

## 📚 Topics Covered
### 🔹 String Fundamentals
- String immutability
- Memory management (String Pool)
- String comparison (== vs .equals())
### 🔹 String Functions
length(), charAt(), compareTo()
equals(), equalsIgnoreCase()
toUpperCase(), toLowerCase()
### 🔹 Substring
- Extracting substrings using substring()
- Index-based operations
### 🔹 Interning Concept
- String constant pool
- Memory optimization using interning
### 🔹 StringBuilder
- Mutable strings
- Efficient concatenation
- Performance advantage over String

## 💻 What I Practiced
### 🔸 Basic Problems
- Check if a string is palindrome
- Find the largest string (lexicographically)
- Print all alphabets (a–z)
### 🔸 Intermediate Problems
- Convert first letter of each word to uppercase
- Find shortest path from directions (N, S, E, W)
### 🔸 Advanced Problems (High Value)
- String Compression (VVI)
- Optimizing repeated characters
- Efficient traversal using StringBuilder
